# Java OOP and JavaFX Applications (CV6 & CV7)

This project contains Java and JavaFX applications created as part of university coursework, focused on object-oriented programming and GUI development.

## Description

The project combines backend logic (OOP) with graphical user interface (JavaFX).

It includes:

- student data management using object-oriented design
- interactive GUI applications built with JavaFX
- hierarchical and tabular data visualization

## Features

### Object-Oriented Programming (OOP)
- working with custom classes and objects
- encapsulation of data (e.g. Student class)
- data manipulation through methods
- basic data structures and collections
- separation of logic and UI
- researching and analysis game files

### Student Management Application (CV6)
- TableView for displaying structured data
- editable table cells (name, surname, year)
- adding and removing records
- dynamic updates of UI
- data visualization using PieChart
- connection between data model and UI

### TreeView Application (CV7)
- hierarchical data representation using TreeView
- parent-child structure (categories, items)
- custom icons for nodes
- dynamic creation and modification of elements
- handling user interaction (selection, expansion)

## Technologies

- Java
- JavaFX
- Object-Oriented Programming (OOP)

## What I learned

- applying OOP principles in real applications
- working with JavaFX components (TableView, TreeView, PieChart)
- handling user input and UI events
- connecting backend logic with GUI
- building structured and hierarchical data
